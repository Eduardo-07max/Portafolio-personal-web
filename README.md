# 🚀 Eduardo Quevedo | Full Stack Developer Portfolio

Bienvenido a mi portafolio profesional. Este proyecto es una Single Page Application (SPA) diseñada para mostrar mi trayectoria, habilidades técnicas y los proyectos que he desarrollado como Ingeniero en Sistemas.

## 🛰️ Vista Previa
Puedes ver el proyecto en vivo aquí: [TU-LINK-DE-VERCEL-AQUÍ](https://tu-proyecto.vercel.app)

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** Angular 21, TypeScript, CSS3 (Custom Properties & Flexbox/Grid).
*   **Herramientas de Desarrollo:** Git, GitHub, Vercel (Deployment).
*   **Diseño:** UI/UX enfocado en legibilidad y temática espacial.

## 🌌 Características Principales

- **Diseño 100% Responsive:** Optimizado para dispositivos móviles, tablets y escritorio.
- **Arquitectura de Componentes:** Uso de las últimas funcionalidades de Angular (Signals, Control Flow Syntax `@for`, `@if`).
- **Sección de Trayectoria:** Línea de tiempo detallada con experiencia docente y técnica.
- **Arsenal Tecnológico:** Visualización de stack técnico incluyendo Laravel, MySQL y DaVinci Resolve.

## 📂 Estructura del Proyecto

```text
src/
├── app/

🚀 Instalación y Uso Local
Si deseas clonar este proyecto y ejecutarlo en tu máquina:

1- Clonar el repositorio:

git clone [https://github.com/tu-usuario/tu-repositorio.git](https://github.com/tu-usuario/tu-repositorio.git)

2- Instalar dependencias:

npm install

3- Correr servidor de desarrollo:

ng serve

4. Abre tu navegador en `http://localhost:4200/`
